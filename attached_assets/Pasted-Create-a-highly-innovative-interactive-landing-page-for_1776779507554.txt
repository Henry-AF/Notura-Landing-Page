Create a highly innovative, interactive landing page for a SaaS product called "Notura".

Core concept:
Transform chaos (meetings) into clarity (structured insights) automatically.

The entire page must feel like a live system working in real time as the user scrolls.

Design style:
- Futuristic, minimal, clean UI
- Soft gradients (purple/blue tones)
- Glassmorphism + subtle shadows
- Smooth micro-interactions
- Inspired by Apple + Linear + Notion AI

Scrolling behavior:
- The page is a storytelling experience
- Each section transitions smoothly into the next
- Animations are triggered by scroll (parallax + progressive transformation)

-----------------------------------
